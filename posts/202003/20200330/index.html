任意のバージョンの Hugo をインストールする

Posted at — Mar 30, 2020

自宅のPCをクリーンインストールした後，Hugoをインストールしたらデフォルトで0.40.1が入ってしまい，今使っているテーマは対応してないと言われた．

任意のバージョンをインストールする．

環境

Ubuntu 18.04 LTS

以下のコマンドでは0.40.1がインストールされる

sudo apt install hugo

任意のバージョンをインストールには

https://github.com/gohugoio/hugo/releases で配布されているdebパッケージをダウンロードして，

sudo apt install ./hugo_0.62.2_Linux-64bit.deb

でOK.（上記はv0.62.2をインストール）
